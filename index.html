Projects

Brain Tumor Classification

The BrainTumorApp is a Flask-based web application designed to assist in the diagnosis of brain tumors using MRI scans. It employs a machine learning model, called AChorDS-LVQ, which includes images categorized into four classes: glioma, meningioma, pituitary tumor, and no tumor.

Key Highlights:

  • High Accuracy: The model achieves approximately 99.7% accuracy in classifying brain tumors.
  • Model Explainability: Incorporates explainability features to help users understand the model's predictions.
  • Web Interface: Built with Flask, the application provides a user-friendly interface for uploading MRI images and viewing predictions.

Legal case classification

The Legal Case Predictor App is a Flask-based web application designed to assist legal professionals in analyzing and classifying legal cases. By leveraging machine learning techniques, the app predicts whether a case pertains to housing or eviction based on the textual information provided.

Key Highlights:

  • Flask Web Application: Provides a user-friendly interface for uploading legal case documents and viewing classification results.
  • Explainable AI: Visualizes the most influential words from the models, enhancing transparency and interpretability of predictions.
  • Low Computational Cost: In contrast to transformer-based approaches, it achieves high accuracy with minimal computational resources. It does not need GPU for training.
